# HOUSE

### Summary
**This project is still under development and will be completed soon. Star it & check back later!**

**HOUSE** is a casino programmed entirely in C++

Implemented Casino Games:
* Casino War
    - 1:1 payout

* Horse Race
    - 5:1 payout
